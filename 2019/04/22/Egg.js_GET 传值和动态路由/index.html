Egg.js_GET 传值和动态路由

发表于 2019-04-22 更新于 2019-05-01 分类于 Node.js
本文字数： 1.2k 阅读时长 ≈ 1 分钟

GET 传值

获取 URL 路径中的 GET 传值。

测试请求：<http://127.0.0.1:7001/?name=andy&password=123456
页面显示：{“name”:”andy”,”password”:”123456”}

动态路由

根据不同的 URL 路径目录，响应不同的值。

测试请求：http://127.0.0.1:7001/list/123
页面显示：123

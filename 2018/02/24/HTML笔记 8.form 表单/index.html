表单

  • form：表示HTML页面上的表单；

  • input：输入文本框，用于收集用户输入的数据；

  • button：表单中的按钮；

  • method：get/pust

  • action：发送表单数据到指定的服务器位置；

  • enctype：表单数据的编码方式

    | enctype值 | 说明 |
    | ——————————— | ———— |
    | application/x-www-form-urlencoded | 默认值 |
    | multipart/form-data | 用于上传文件 |
    | text/plain | |

accesskey 属性

accesskey 作用：设定一个或几个用来选择页面上元素的快捷键。

效果就是，在 Mac 的 Safari 浏览器上同时按：control+option+accesskey 就可以将焦点跳转到指定的 inpput 输入框。

tabindex 属性

作用：HTML页面上的键盘焦点可以通过按Tab键在各个元素之间进行切换。用tabindex 属性可以改变默认的转移顺序。

perfops-cli - 一款IT运维工具

GitHub：perfops-cli

PerfOps CLI 是一款适合于运维人员使用的命令行工具，方便从多地点执行 ping、traceroute、mtr 等网络基准测试。
PerfOps 使用简单，你可以从位于全世界的数百个节点向目标地址发起网络测试。在不设置 API 密钥的条件下，它允许每小时进行 10 次测试，免费注册用户则可以在每小时内发起最多 200 次测试。

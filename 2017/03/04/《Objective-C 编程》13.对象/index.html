《Objective-C 编程》13. 对象

发表于 2017-03-04 更新于 2019-05-01 分类于 Objective-C 编程 Valine：
本文字数： 925 阅读时长 ≈ 1 分钟

框架

  • 框架（famework）：是由许多相关的类、函数、常量以及数据类型组成的库。
  • 编写 Objective-C 程序时，要使用 Foundation 框架，该框架包含了很多基础类。

对象

类

  • 类（class） 负责描述某个特定类型的 对象（object）；
  • 类 可以描述抽象概念的对象，也可以描述实际存在的事物；
  • 一个类定义了一种对象，同时它也可以用来创建这种对象。

#import & #include

#import <Foundation/Foundation.h>

  • #import 指令导入更快更有效率；
  • #include 指令告诉编译器做呆板的复制粘贴，将包含的内容粘贴到指定的目标文件夹中来；
  • #import 指令会让编译器先检查之前是否已经导入过这个文件，或是已经被包含到目标文件夹中了。

创建并使用对象

  • NSDate *now: 声明了一个数据类型为 NSDate * 的指针变量 now ，这个指针保存了 NSDate 实例在内存中的地址；
  • [NSDate date]: 消息发送语句，向 NSDate 类发送了 date 消息，让它执行 date 方法；
  • 消息发送语句：[接收方 选择器]；
  • date 方法执行后，NSDate 类会在堆（heap）上给 NSDate 实例声明一部分内存（去堆上申请一块连续的内存，称之为缓冲区（buffer）），将 NSDate 实例初始化为当前的日期／时间，然后返回新对象的地址。
  • 发送 timeIntervalSince1970 给 now 变量 指向的 NSDate 实例，返回一个双精度数（double）保存在了变量名为 seconds 中。
  • date 是一个 类方法，通常，类方法会创建类的实例，并初始化实力变量；
  • timeIntervalSince1970 是一个 实例方法，实例方法会提供实例中实例变量的信息，或是对实例的实例变量进行操作。

Objective-C 语言命名习惯

  • Objective-C 是区分大小写的；
  • 实例变量和方法都采用 “骆驼拼写法” 命名；
  • 类的名称要加前缀，以大写字母开头；
